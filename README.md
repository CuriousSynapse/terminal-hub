# TerminalHub

A dark, terminal-inspired personal productivity dashboard. Manage to-dos with due dates, priorities, and folders — and create custom sections to link out to any files, apps, or URLs you use.

## Features

- **To-Do List** — Tasks with due dates, priorities (High/Normal/Low), progress bar, and relative time display ("today", "tomorrow", "in 5d")
- **Quick Dates** — One-click buttons for Today, Tomorrow, This Week, Next Week
- **Folders** — Organize tasks into folders (Inbox, Work, Personal, etc.)
- **Search & Filter** — Filter by status (Active/Completed/Overdue) and search by text
- **Custom Sections** — Create your own sections (Finance, Fitness, Meals, etc.) as link collections to local files or URLs
- **Smart Link Icons** — Auto-detects file types (.xlsx → 📊, .pdf → 📄, URLs → 🌐)
- **Export/Import** — JSON backup and restore
- **Keyboard Shortcuts** — `N` for new task, `/` for search
- **Fully Local** — All data stored in localStorage, no server needed

## Usage

1. Download or clone this repo
2. Open `index.html` in your browser
3. Start adding tasks and creating sections

No build step, no dependencies, no server required.

## Tech Stack

- Vanilla HTML/CSS/JS
- JetBrains Mono + Space Grotesk fonts (Google Fonts)
- localStorage for persistence
